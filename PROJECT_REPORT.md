### 2.3 قاعدة البيانات (Prisma Schema)

```prisma
model Product {
  id          String   @id @default(cuid())
  name        String              // اسم المنتج
  description String              // الوصف
  price       Float               // السعر (ليرة سورية)
  image       String              // مسار الصورة
  category    String              // التصنيف
  options     String   @default("[]")  // خيارات المنتج
  isActive    Boolean  @default(true)   // حالة الظهور
  sortOrder   Int      @default(0)       // ترتيب العرض
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt
}
```

#### مكونات الصفحة الرئيسية:

##### 3.1.1 Header (شريط التنقل)
- **الملصقات (Badge)**: يختفي عند التمرير للأسفل
- **الخلفية**: شفافة ثم تصبح navy/90 عند التمرير
- **الشعار**: صورة دائرية مع تأثير توهج ذهبي
- **روابط التنقل**: المنتجات، عن بريڤيا، تواصل معنا
- **زر CTA**: "اطلب الآن" مع أيقونة سلة
- **قائمة الموبايل**: hamburger menu متحركة

##### 3.1.2 Hero (قسم البطل)
- **الخلفية**: تدرج navy مع تأثيرات جسيمات متحركة
- **الشعار**: صورة دائرية 144x144px مع ظل ذهبي
- **العنوان**: BRIVIA بحجم 5xl-7xl مع تأثير التوهج
- **الوصف**: نص وصفي بطول 60 حرف
- **الأزرار**: 
  - "استكشف المجموعة" - تدرج ذهبي
  - "تواصل معنا" - حدود ذهبية مع أيقونة واتساب
- **المميزات**: 3 ملصقات صغيرة

##### 3.1.3 ProductGrid (شبكة المنتجات)
- **العنوان**: "مجموعتنا الفاخرة"
- **التصنيفات**: أزرار قابلة للتحديد (الكل + الفئات)
- **الشبكة**: 3 أعمدة على الشاشات الكبيرة
- **البطاقات**: تأثير 3D عند التمرير
- **نافذة الطلب**: modal متحركة لطلب المنتج

##### 3.1.4 AboutSection (قسم عن الشركة)
- **التخطيط**: عمودين (صورة + نص)
- **الصورة**: parallax effect مع إحصائيات عائمة
- **المحتوى**: وصف + 4 مميزات
- **الأزرار**: تصفح المنتجات + تواصل معنا

##### 3.1.5 Footer (الذيل)
- **الأقسام**: 3 أعمدة
  - الشعار والوصف
  - روابط سريعة
  - معلومات التواصل

- `shadow-gold`: `0 0 20px rgba(201, 169, 97, 0.3)`

- `shadow-soft`: `0 4px 20px rgba(11, 26, 61, 0.08)`
- `shadow-soft-lg`: `0 8px 40px rgba(11, 26, 61, 0.12)`

#### التدرجات (Gradients):
- `bg-gradient-radial-gold`: من المركز للحواف
- `text-gold-gradient`: للنص الذهبي

- `animate-pulse`: نبض خفيف

## 6. الأمان والمصادقة

### 6.1 نظام JWT

```typescript
// Token Structure (jose library)
{
  "role": "admin",
  "iat": 1699900000,      // Issued at
  "exp": 1700504800       // Expiration (7 days)
}
```

### 6.2 Middleware

```typescript
// middleware.ts
- /admin/* → يتطلب token صالح
- /admin/login → مسموح للجميع
- /api/admin/* (عدا login) → يتطلب token صالح
```
